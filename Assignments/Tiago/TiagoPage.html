Lab class 1 - Good and bad design

Good example

Interface Objective

Calendário digital, ver o dia corrente nos calendários


Aspects

Normalmente todos os calendários digitais tem o dia corrente visualmente destacado, para uma rápida identificação por parte do usuário do dia em que se encontra




Bad example

Interface Objective

Intuito de ligar conectores USB às respetivas portas


Aspects

O utilizador tem de olhar para ver qual a forma correta de ligar ambos os componentes, ou experimentar até conseguir


Design suggestion

Colocar uma seta chamativa na porta e no conector para saber a forma correta de encaixe
